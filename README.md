# react-native-template

# 介绍

此库封装了常用 UI 组件和部分项目配置，可用于快速迭代开发基于 RN 的项目。

# 已完成

- Container
- Error
- Image
- Loading
- Navigation
- Touchable(Button)
- List
- Row
- Toast
- StorageManager
- Alert
- ActionSheet
- Form
- HighlightText
- Picker
- Overlay
- Badge
- Theme 支持切换主题

# 待完成
- Card
- PermissionsManager
- Popover
- Segmented
- Banner
- Stepper
- PayManager
- MediaPickerManager
- Services 结构

## Installation

```
$ npm install
$ cd ios && pod install
$ npm run ios // npm run android

```

# react-native-template

# 介绍

此库封装了常用UI组件和部分项目配置，可用于快速迭代开发基于RN的项目。

# 已完成

- Container
- Error
- Image
- Loading
- Navigation
- Touchable(Button)
- List
- Row
- Toast
- StorageManager
- Alert
- ActionSheet
- Form
- HighlightText
- Picker
- Overlay

# 待完成

- PermissionsManager
- Segmented
- Theme重新设计，支持切换主题
- Card
- Banner
- Stepper
- PayManager
- MediaPickerManager
- Services结构

## Installation

```
$ npm install 
$ cd ios && pod install
$ npm run ios // npm run android

```
